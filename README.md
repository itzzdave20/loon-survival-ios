# Loon Survival

A Python/Kivy raycaster game with an installable browser/PWA build.

## Run the web app on Ubuntu Linux

```bash
sudo apt update
sudo apt install -y python3
./run_webapp.sh
```

Open:

```text
http://localhost:8000/webapp/
```

On another device on the same Wi-Fi, open:

```text
http://YOUR-UBUNTU-IP:8000/webapp/
```
